# 📊 Git Insights

> **What do your commits say about you?** Analyzing popular open source repos with [GitScope](https://github.com/promptpolish-ai/gitscope).

## Why This Exists

Your git history tells a story. How often do you commit? When are you most productive? Who contributes the most? 

This repo is a collection of Git repository analyses of popular open source projects — generated with [GitScope](https://github.com/promptpolish-ai/gitscope), a free CLI tool that turns any git repo into a beautiful interactive HTML report.

## Run Your Own Analysis

Want to analyze YOUR repository? It takes 30 seconds:

```bash
npx gitscope /path/to/your/repo --open
```

No install needed. Just run the command and your browser opens with a full report.

## What a GitScope Report Shows

- **📈 Contribution Timeline** — Weekly activity
- **🕐 Hourly Heatmap** — When you code most
- **📅 Day of Week** — Work-life balance
- **👥 Contributors** — Per-author stats
- **📁 Hot Files** — Most-changed files
- **🏆 Streaks** — Longest consecutive days
